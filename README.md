# 公司資訊快速查

快速在面試趣、GoodJob、Dcard 等網站搜尋公司資訊。

## Features

-   開啟職缺或公司頁面後，自動擷取公司名稱，點擊擴充功能即可快速開啟搜尋頁面。

    ![image](./demo.gif)

-   右鍵選單快速搜尋。
